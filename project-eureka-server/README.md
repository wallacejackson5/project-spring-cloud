# Getting Started

This project was built with [Spring cloud](https://spring.io/projects/spring-cloud).

## Available Scripts

In the project directory, you can run:
- `make up`

## Additional information
Open [http://localhost:8001](http://localhost:8001/project-eureka-server/) to view it in the browser.

You need to put the username and password.
