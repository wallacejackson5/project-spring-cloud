# Getting Started

This project was built with [Spring cloud](https://spring.io/projects/spring-cloud).

## Available Scripts

In the project directory, you can run:

### `make up`

Runs the app in the development mode.

You need to put the username and password.

### `npm run build`

Builds the app.
